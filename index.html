Welcome

My name is Brandon Seymour, and this ePortfolio represents my work throughout the Computer Science program at Southern New Hampshire University. My path to finishing my degree has not been a straight one. I started college after serving in the Marine Corps, changed majors, moved states, took time away from school, and eventually came back to finish what I started.

This portfolio focuses on my CS330 Computational Graphics project, which I enhanced across software design and engineering, algorithms and data structures, and databases. My long-term goal is to become a Software Engineering Manager, so I wanted this portfolio to show not only that I can write code, but that I can improve, organize, explain, and grow a project over time.
