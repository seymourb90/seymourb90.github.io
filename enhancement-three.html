Enhancement Three: Databases

This enhancement focuses on transforming the original CS330 Computational Graphics project into a data-driven application through the use of external CSV data. Object information is now loaded from a CSV file rather than being hardcoded within the application.

This approach improves maintainability and allows scene modifications without changing source code.
